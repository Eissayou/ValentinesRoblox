## üéÆ What It Is

An interactive experience where her dog Coco delivers the Valentine's message. Walk up to Coco, press E, and go through the dialog to get to the question. If you try to click "No", the button dodges you (and eventually gives up after 12 attempts). Click "Yes" and the whole room turns into a Gangnam Style party.

## ‚ú® Features

- **Valentine venue** - 100x150 stud room with pink/red decorations, floating hearts, and a red carpet
- **Coco the dog** - Her actual Shih-Poo as a 3D model (scaled 1.5x)
- **Dialog system** - Typewriter effect, click to advance through the message
- **Dodging "No" button** - Moves and shrinks each time you hover over it
- **Gangnam Style celebration** - Dancers doing the actual dance, two-track music sequence, fireworks, camera shake, post-processing effects

## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "ValentinesRoblox.rbxlx"
```

Next, open `ValentinesRoblox.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
